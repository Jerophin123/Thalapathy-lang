# THALAPATHY Development Roadmap

## Phase 1 — Minimal core (v0.1.0) [CURRENT]
- AST tree-walking interpreter.
- Core type check resolver.
- Caret diagnostics.
- Custom binary THALA IR compiler and serializer.
- REPL interactive loop.

## Phase 2 — Language extensions
- Standard collection method closures.
- Class properties and multiple interfaces (`kaththi`).
- Dynamic operator overloading.

## Phase 3 — High-performance Compiler
- Custom VM engine execution loop for bytecode targets (`.thalac`).
- Native compilation and LLVM integration.
