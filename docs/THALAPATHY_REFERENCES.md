# THALAPATHY Cinema Reference Catalogue

This document catalogs references to Thalapathy Vijay's filmography, dialogue style, and fan culture as represented in keyword names and compilation messages.

## Filmography & Keyword Alignments
- **Ghilli (2004)**: Used as the conditional keyword `ghilli` (replaces `if`) due to the high-stakes, fast-paced choice dynamics in the film.
- **Pokkiri (2007)**: Map keyword `pokkiri` (replaces `try`). Translates to "daring" attempt in execution.
- **Kaavalan (2011)**: Keyword `kaavalan` (replaces `catch`). Represents the protector or guardian capturing threw exception objects.
- **Theri (2016)**: Exception trigger `theri` (replaces `throw`). Refers to an explosive action event.
- **Mersal (2017)**: Declares a function block `mersal` (replaces `fn`). Highlights complex signature definitions.
- **Master (2021)**: Class definition keyword `master` (replaces `class`).
- **Varisu (2023)**: Inheritance indicator `varisu` (replaces `extends`). Refers to standard family lineage properties.
- **Leo (2023)**: Protected access keyword `leo` (replaces `protected`). Refers to a protected/hidden identity.
- **Bigil (2019)**: Traditional loop construct `bigil` (replaces `for`).
- **Vaathi (Nanban, 2012)**: range loop iterator `vaathi` (replaces `foreach`).

## Dialogue Wordplay Examples
- *"I'm waiting..."*: Reserved future keyword representing async-await operations.
- *Thiruppi*: Keyword `thiruppi` (replaces `return`). Literal translation to "give it back" or "return".
- *Sollu*: Keyword `sollu` (replaces `print`). Literal translation to "tell/output".
- *Uruthi*: Keyword `uruthi` (replaces `const`). Represents a "firm/unchanging" constant choice.
